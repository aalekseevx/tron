Условие задачи

ограничение по времени на ход: 0.5 с
ограничение по памяти на ход: 256 МБ
ввод: стандартный ввод
вывод: стандартный вывод
Игровое поле представляет из себя матрицу размером $$$n \times m$$$ клеток. Всего в игре $$$10$$$ игровых полей. $$$5$$$ из них доступны вам для проведения дуэлей, остальные останутся вам неизвестны. Турниры будут проходить на всех $$$10$$$ полях. В начальный момент времени игроки находятся на своих стартовых позициях. Стартовые позиции могут отличаться на разных картах. Игроки ходят по очереди. Каждый ход (здесь и далее, ход - действие одного игрока) они могут перейти в соседнею по стороне клетку, которая не была посещена кем либо ранее и в которой не содержится препятсвия.
За каждую посещённую клетку игрок получает $$$1$$$ очко. За каждую посещённую клетку с монетой игрок получает дополнительно $$$2$$$ очка.

Также на поле в заранее определённых местах находятся бонусы двух типов

  • Ускоряющий бонус увеличивает вашу скорость в $$$2$$$ раза на $$$20$$$ ходов
  • Замедляющий бонус уменьшает вашу скорость в $$$2$$$ раза на $$$20$$$ ходов
Если ваша скорость в $$$x$$$ раз больше (меньше) скорости противника, то право хода вам будет предоставляться в $$$x$$$ раз чаще (реже).

Если игрок не может сделать ход, то игра для него заканчивается, но противник продолжает ходить. Игра заканчивается, когда оба игрока не могут сделать ход.

Цель игры - заработать как можно больше очков.
Входные данные
Для чтения входных данных на запросы программа должна использовать стандартный ввод.

В первой строке входных данных записана строка $$$'RED'$$$ или строка $$$'BLUE'$$$. Это цвет фишки вашего игрока. Красная фишка ходит первой.


Во второй строке записано число $$$k$$$ от $$$1$$$ до $$$10$$$ - номер карты, на которой вы играете.


В третьей строке записано $$$2$$$ целых числа - $$$n$$$ и $$$m$$$.


Затем, пока у вас есть доступные ходы действует слудующий протокол взаимодействия:
На вход подаётся матрица символов, разделённых пробелами из $$$n$$$ строк и $$$m$$$ столбцов. Это игровое поле для которого вам необходимо вывести ход.

Условные обозначения:

  • $$$.$$$ - свободная клетка
  • $$$R$$$ - клетка с фишкой красного игрока
  • $$$B$$$ - клетка с фишкой синего игрока
  • $$$r$$$ - клетка, посещённая красным игроком
  • $$$b$$$ - клетка, посещённая синим игроком
  • $$$X$$$ - клетка с препятствием
  • $$$C$$$ - клетка с монетой
  • $$$U$$$ - клетка с ускоряющим бонусом
  • $$$D$$$ - клетка с замедляющим бонусом

В тот момент, когда у вас не будет доступных ходов, ваша программа будет остановлена. Мы гарантируем: каждый раз, когда программе на вход поступило поле у неё есть хотя бы один возможный вариант хода.
Выходные данные
Для записи выходных данных на запросы программа должна использовать стандартный вывод. Выведите один символ из множества $$$\{ L, R, U, D \}$$$ и последующий перенос строки. После вывода каждой строки программа должна выполнить операцию $$$flush$$$. Условные обозначения:
  • $$$L$$$ - ход влево
  • $$$R$$$ - ход вправо
  • $$$U$$$ - ход вверх
  • $$$D$$$ - ход вниз

#include

using namespace std;

mt19937 rnd(time(0));

int get_rnd(int r) {
    return abs(int(rnd())) % r;
}

int level, n, m;
string player;

void make_move() {
    vector> field(n, vector(m));
    int my_x = -1, my_y = -1;
    char my_ptr = player == "RED" ? 'R' : 'B';

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            cin >> field[i][j];
            if (field[i][j] == my_ptr) {
                my_x = i;
                my_y = j;
            }
        }
    }

    int xc[] = {0, 1, 0, -1};
    int yc[] = {-1, 0, 1, 0};

    vector possible_moves;
    char lit[] = {'L', 'D', 'R', 'U'};
    char good[] = {'.', 'C', 'U', 'D'};

    for (int move = 0; move < 4; move++) {
        int nx = my_x + xc[move];
        int ny = my_y + yc[move];
        if (nx >= 0 && nx < n && ny >= 0 && ny < m) {
            int ok = 0;
            for (char c: good) {
                ok |= (field[nx][ny] == c);
            }
            if (ok) {
                possible_moves.push_back(move);
            }
        }
    }

    cout << lit[possible_moves[get_rnd(possible_moves.size())]] << endl;
    cout.flush();
}
int main() {
    cin >> player >> level >> n >> m;
    while(true) {
        make_move();
    }
}
